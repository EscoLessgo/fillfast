Privacy Policy

Last updated: December 13, 2025

1. Data Collection

We collect minimal data necessary to function as a Discord Activity:

  • Discord User ID & Username: To display your identity in-game and manage game sessions.
  • Game State: Temporary data about your current game moves and score.

2. Data Usage

This data is used solely for:

  • Real-time multiplayer gameplay.
  • Session management (connecting you to the correct room).

We do not sell, trade, or otherwise transfer your personally identifiable information to outside parties.

3. Data Retention

All game data is ephemeral. It is stored in volatile memory on our game server and is strictly deleted once the game session ends or the room is closed.

4. Contact

If you have questions, please reach out via our support channel on Discord.
